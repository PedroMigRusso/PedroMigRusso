## 🏗️ Featured Projects

### 🏥 [BetterHealthCare System](https://github.com/PedroMigRusso/BetterHealthCare)
Full-stack healthcare data platform addressing the lack of patient data synchronisation between Portugal's public and private healthcare providers.
Built across three independent layers all consuming the same REST API:
- **[REST API Backend](https://github.com/PedroMigRusso/BetterHealthCare)** — .NET Core 8, Entity Framework Core, SQL Server, Repository Pattern, Unit Tests
- **[ASP.NET Core MVC Frontend](https://github.com/PedroMigRusso/BetterHealthCare_WebApp)** — Razor Views, Bootstrap, HttpClient
- **[React SPA](https://github.com/PedroMigRusso/BetterHealthCare_ReactApp)** — React 18, TypeScript, Vite, Axios, Tailwind CSS

### 👥 [CollaboratorsManagement](https://github.com/PedroMigRusso/CollaboratorsManagement)
ASP.NET Core MVC web application with full CRUD, pagination, filtering, and a clean layered service architecture. Built with .NET 8, Entity Framework Core, and SQLite.
